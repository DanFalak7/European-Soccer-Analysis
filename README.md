# European-Soccer-Analysis Using SQL
# Project Overview
This project analyzes historical match data from Europe’s Top 5 football leagues (Premier League, La Liga, Serie A, Bundesliga, Ligue 1) using SQL. The goal is to uncover patterns in competitiveness, team dominance, scoring trends, and home advantage across multiple seasons. The analysis moves beyond basic league tables by aggregating data across seasons, ranking teams and match stages, and benchmarking performance against league-wide baselines.
